# 项目目标

开发一个无需联网运行的桌面软件，用于帮助用户健康使用电脑，自动提醒肩颈放松训练，并通过摄像头实时检测姿态质量。

核心目标：

* 启动后自动检测肩颈姿态
* 引导做 1 分钟标准肩颈活动
* 每 30 分钟提醒一次
* 记录电脑使用时长
* 记录姿态评分数据
* 生成周报统计
* 支持最小化到托盘
* 支持开机自启动

# 核心功能设计

## 1️⃣ 启动流程

1. 应用启动
2. 自动启动本地 Python 服务
3. 申请摄像头权限
4. 引导用户做 1 分钟标准肩颈活动
5. MediaPipe 实时检测
6. 姿态评分
7. 若不达标 → 语音提醒 + 动画演示

---

## 2️⃣ 姿态检测逻辑

使用 MediaPipe Pose 检测：

关键点：

* 左右肩
* 左右耳
* 脊柱中线
* 颈部倾斜角度
* 肩部高度差
* 头部前倾角

计算指标：

* 头部前倾角度 > 20° 判定驼背
* 肩部高度差 > 阈值 判定耸肩
* 身体倾斜角 > 阈值 判定歪斜

姿态评分算法：

score = 100 - (驼背扣分 + 耸肩扣分 + 倾斜扣分)

## 3️⃣ 每 30 分钟提醒机制

使用后端调度器：

* 弹出训练引导窗口

## 7️⃣ AI 个性化建议

当连续 3 次姿态低于 60 分：

shoulder_diff: 12,
spine_angle: 15,

}

* 桌椅高度调整

# 错误兜底逻辑

1. 摄像头不可用 → 弹出手动训练模式
2. MediaPipe 加载失败 → 切换纯提醒模式
3. AI 调用失败 → 使用本地规则建议
4. Python 服务异常 → 自动重启机制

# Acceptance Criteria (AC)

### AC-1 启动检测

* 应用启动 5 秒内完成初始化
* 摄像头成功打开
* 显示姿态骨架

### AC-2 姿态识别准确

* 驼背识别准确率 > 85%
* 肩部倾斜识别准确

### AC-3 定时提醒

* 每 30 分钟 ± 1 分钟误差提醒
* 支持用户自定义时间

### AC-4 数据持久化

* 重启后数据完整
* 周报正确生成

### AC-5 离线运行

* 无网络可完整使用
* AI 功能自动降级

### AC-6 性能

* CPU 占用 < 20%
* 内存 < 300MB
